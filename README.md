# ChatMate NLP Chatbot

ChatMate is an NLP-based desktop chatbot application developed using Python, Tkinter, and NLTK.  
The chatbot understands user messages and responds intelligently using trained intents from a JSON dataset.

---

## Features

- Desktop chatbot using Tkinter GUI
- NLP-based intent recognition
- Custom chatbot responses
- Easy training using intents dataset
- Beginner-friendly Python project
- Fast and lightweight application

---

## Technologies Used

- Python
- Tkinter
- NLTK
- JSON
- Pickle
